# 🤖 AI Hand Tracking System

This project uses Computer Vision to control a computer using hand gestures.

## 🚀 Features
- ✋ Hand Tracking
- 🔢 Finger Detection
- 🖱️ Virtual Mouse
- 🎨 Air Drawing
- 🖥️ GUI App

## 🛠️ Tech Stack
- Python
- OpenCV
- MediaPipe

## ▶️ How to Run
pip install -r requirements.txt  
python gui_app.py
